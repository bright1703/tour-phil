## Почему все едут в Эль-Нидо

Потому что такого нигде больше нет. Известняковые скалы высотой 300 метров, упавшие в бирюзовое море. Скрытые лагуны с водой такого цвета, что не верится. Пляжи без людей.

Эль-Нидо предлагает четыре стандартных маршрута — A, B, C, D. Каждый охватывает разные точки.

## Тур A — Классика

Самый популярный и самый рекомендуемый для первого дня.

- **Большая лагуна** — внутренняя лагуна за скалами, снорклинг у рифов
- **Малая лагуна** — можно войти только на каяке или вплавь
- **Секретная лагуна** — через узкую щель в скале
- **Пляж 7 Коммандо** — для обеда и отдыха
- **Остров Симизу** — снорклинг с черепахами

## Тур B — Пещеры

- **Пещера Кодугнон** — Cathedral Cave с летучими мышами
- **Остров Энатула** — белый песок, пальмы, никакой цивилизации
- **Остров Пинагбуютан** — вид-открытка Эль-Нидо

## Тур C — Скрытые места

- **Секретный пляж** — буквально спрятан за скалами
- **Остров Вертолет** — необитаемый, для снорклинга
- **Святыня Матинлок** — внутренняя лагуна с дикой природой

## Тур D — Для спокойного отдыха

- **Малая лагуна** — менее посещаемый маршрут
- **Лагуна Кадлао** — спокойная, с манграми
- **Пляж Натнат** — для обеда среди скал

## Комбо-тур A+C

Самый полный маршрут — для тех, кто хочет увидеть лучшее из двух программ за один день. Длиннее обычного, выходим раньше.
